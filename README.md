## 진행방식
  - 브랜치 파셔서 문제풀이 후 풀리퀘를 날려주세요(코드에 주석으로 풀이과정 붙여주세요)<br>
  - 필수:매주 공통문제 4개 개별문제 3개 풀어오기(개별문제는 프로그래머스lv1 외의 문제 부탁드리겠습니다)<br>
  - 선택:각자 블로그에 2주에 한번 알고리즘/자료구조 개념 정리해서 올리도록 합시다(ex:dfs/bfs,dp,그리디,stack/queue 등등)<br>
  - 마감기한은 토요일
  - 벌금은 문제당 3천원!
